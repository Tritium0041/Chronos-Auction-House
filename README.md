# 时空拍卖行：*立即开启你的时空倒爷之旅！*

 这是一款支持 3-5 人的半合作经济博弈桌游，玩家作为穿梭时空的古董商，需要在激烈的**双模式竞标**与**动态市场操盘**中掠夺珍宝，同时时刻警惕全员共享的**时空稳定性**，防止宇宙崩塌导致全盘皆输。

> 倒计时 3-2-1，摊开手掌。你以微弱的优势赢下了这张【戴森球蓝图】，但还没来得及庆祝，警报声刺破了拍卖行的宁静——时空稳定性跌破了 15% 的红线！
> 
> 市场陷入恐慌，所有的文物价格强制暴跌。下一张翻开的事件卡是【时空震荡】，持有高消耗文物的人将面临惩罚。此刻，所有人必须立即做出抉择：是捐出巨款修复时空，挽救大家的游戏？还是看着世界毁灭，赌自己手里那枚稳定锚能让你成为废墟上唯一的赢家？
> 
- 拒绝单一机制！体验**公开叫价**的豪气与**密封竞标**的尔虞我诈。你是选择用金钱碾压对手，还是在迷雾中通过心理战险中求胜？
- 没有任何价格是永恒的。通过**事件卡**与**投票**，你可以让手中的 “垃圾” 一夜暴富，也能让对手囤积的 “珍宝” 瞬间贬值。
- 贪婪是有代价的。当全员共享的时空稳定性归零，只有持有**稳定锚**的人才能幸存。你会选择牺牲利益修复时空，还是独善其身赌一把末日生存？

总体来说 这个游戏是博弈点很充足的，最终游戏的名次由文物价值和名望分共同决定，在争夺文物和围绕时空稳定性方面都可以有博弈的因素。

目前这个游戏还处于很早期的阶段，如果你感兴趣的话，可以带上纸笔，下载git仓库中的规则书和生成好的pdf文件并打印裁剪来进行试玩。目前还没有做中心面板和资金指示物，这一块的内容暂时需要用纸笔记录了。如果你试玩后有任何想法或问题想反馈，非常欢迎建立一个issue来讨论！
